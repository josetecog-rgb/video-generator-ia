# Creador de videos cortos

## Descripción
**1. DESCRIPCION DEL PROYECTO**
El proyecto "Creador de videos cortos" es una aplicación web que utiliza inteligencia artificial (IA) para generar videos virales en varias plataformas. La aplicación debe tener la capacidad de generar temas, guiones de historia, imágenes y videos cortos con audio y efectos. El objetivo es crear contenido de alta calidad que sea viral en plataformas como YouTube, Instagram, Facebook, TikTok, Telegram, Discord y WhatsApp.

**2. STACK TECNOLOGICO**
* Frontend: React o Angular
* Backend: Node.js con Express.js
* Base de datos: MongoDB o PostgreSQL
* IA: Utilizar bi
